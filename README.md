# Students page

Using Vue & Vue Router, set up:

\* A students list component that renders on the Home route.
(Display a 2 column table, one should be full name (firstName + lastName), second column should be student number)

\* Each Student's name in the table should link to an individual page that uses the `id` as the dynamic segment of the route.
(Example: `/student/3`)

\* Each individual page sould display the avatar image, full Name (firstName + lastName), email, phone number and a list of courses (`courseName - courseCode `).

For student data, this is available in `src/data/index.js` import and use as needed.

For styling, [daisyUI](https://daisyui.com/) is installed in this project. 
You can use just straight up Tailwind classes or use the available [components](https://daisyui.com/components/) that daisy has.
